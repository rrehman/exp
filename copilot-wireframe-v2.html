How do I create a payout and track its status in one flow?
C
API Copilot · scoped to Payouts
To create and track a payout, chain two endpoints: POST /payouts to initiate, then GET /payouts/:id to poll status. Here's the TypeScript flow using the SDK:
typescript · SDK 2 endpoints
const payout = await client.payouts.create({
  amount: 5000,
  currency: "USD",
  destination: "ba_1234"
});

// Poll until terminal state
const status = await client.payouts.get(payout.id);
⚡ Want to handle failures automatically too?
I can extend this to add POST /payouts/retry on a failed status — covering the full payout lifecycle in one orchestrated block. This is what the SDK is built for: multi-endpoint flows, not just single calls.

Complete lifecycle — create, poll, and auto-retry on failure:
typescript · SDK 3 endpoints orchestrated
const payout = await client.payouts.create({...});
let status = await client.payouts.get(payout.id);

if (status.state === "failed") {
  // Retry automatically — same idempotency key
  status = await client.payouts.retry(payout.id);
}

return status; // terminal: succeeded | cancelled
